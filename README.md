# ZASHE MANAGEMENT LTD Website

Static Vite + Three.js website for ZASHE MANAGEMENT LTD.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is created in `dist/`.

## Brand Assets

Logo, favicon, app icon and social preview assets are in `public/brand/`.

## Media Note

The construction photography in `public/media/` is synthetic commissioned brand imagery for website presentation. It must not be described as completed client work, evidence of accreditation, or proof of certification.
