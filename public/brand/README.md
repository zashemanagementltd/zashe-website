# ZASHE Brand Assets

These assets are the project-local brand kit for the website.

- `logo-mark.svg` - square Z monogram.
- `logo-horizontal-on-dark.svg` - header/logo for dark backgrounds.
- `logo-horizontal-on-light.svg` - logo for light backgrounds.
- `logo-stacked.svg` - stacked brand lockup.
- `favicon-32.png`, `favicon-192.png`, `favicon-512.png` - browser/PWA icon exports.
- `apple-touch-icon.png` - iOS home-screen icon.
- `og-card.svg`, `og-image.png` - social preview image.

The construction photography in `public/media` is synthetic commissioned brand imagery. It must not be described as completed client work or proof of certification.
